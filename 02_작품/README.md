# 작품

새 작품은 `_작품 템플릿`을 복제해 시작합니다. 참고자료를 이 폴더에 그대로 복사하지 말고, 사용자에게 확정받은 작품 설정만 SSOT 문서에 기록합니다.

## 새 작품 생성 절차

1. `_작품 템플릿` 폴더를 복사합니다.
2. 복사한 폴더명을 작품명으로 변경합니다.
3. `00 작품 홈.md`를 먼저 작성합니다.
4. `01 작품 개요.md`와 `02 핵심 재미와 타깃.md`을 작성합니다.
5. `00_운영/06 현재 작업.md`에 활성 작품으로 등록합니다.
6. 필요한 원문만 `원문 보강/첫작품_필수자료/`에 보강합니다.
7. `08 프롬프트 정본.md`에서 `v0.1.0`을 작성합니다.
8. `11 테스트/테스트 케이스.md`에서 `TEST-0001`부터 실행합니다.
9. 결과를 `테스트 결과 로그.md`에 기록합니다.
10. 수정이 발생하면 `13 변경 기록.md`에 버전과 이유를 남깁니다.

## 운영 규칙

- 작품의 확정 설정은 해당 작품 폴더에만 기록합니다.
- 작품 폴더와 지식베이스가 충돌하면 작품 SSOT를 우선합니다.
- 테스트 결과가 없는 프롬프트는 공개 준비 완료로 표시하지 않습니다.
