# 지식베이스 assets

- `images/`: 원문 이해에 필수적인 보존 이미지
- `attachments/`: 부속 파일

모든 자산은 출처 노트와 상대경로로 연결하고, 첫 작품에 실제 필요한 자료부터 보강합니다.
