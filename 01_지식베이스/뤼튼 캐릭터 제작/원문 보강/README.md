# 원문 보강

첫 작품에서 실제로 채택할 색인 항목의 상세 본문과 필수 이미지만 우선 보존합니다. 약 545개 항목 전체를 선제적으로 옮기지 않습니다.

## 보강 우선순위

1. 상태창과 시스템
2. 프롬프트 구조
3. 예시대화
4. 키워드북
5. 테스트
6. 미디어
7. 공개 전 점검

## 문서 필수 항목

- Notion 출처와 원문 카테고리
- 게시글 번호
- 보존한 핵심 내용
- 작품 적용 가능성
- 주의점과 검증 필요 여부
