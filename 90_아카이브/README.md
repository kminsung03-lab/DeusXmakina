# 아카이브 사용 금지 안내

이 폴더의 자료는 비활성 백업이며 활성 정본이 아닙니다. 이전 AGENTS.md, 중첩 Obsidian 설정, 기존 Obsidian 볼트 복사본과 진입 노트를 보존합니다.

AI와 Codex는 사용자가 명시적으로 복구·비교를 요청하지 않는 한 이 폴더의 자료를 현재 정본, 현재 설정 또는 현재 지식베이스의 근거로 사용하지 않습니다. 검색 결과에 포함되더라도 판단 근거에서 제외하고 비활성 자료라고 표시합니다.

현재 판단 근거는 반드시 다음 활성 경로에서 찾습니다.

- `00_운영/`
- `01_지식베이스/`
- `02_작품/`
- `03_프롬프트_라이브러리/`
- `04_테스트_로그/`
- `05_결정_기록/`
