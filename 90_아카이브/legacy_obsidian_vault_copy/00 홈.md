# 뤼튼 캐릭터 제작 지식베이스

Notion의 **뤼튼 갤러리 정보글 백업**을 Obsidian에서 검색하고 재활용하기 좋게 옮긴 허브입니다.

## 바로 시작

- 새 캐릭터를 만들 때: [[활용 노트/캐릭터 제작 체크리스트]]
- 프롬프트를 설계·수정할 때: [[활용 노트/프롬프트 설계 핵심]]
- 전체 제작 흐름과 상세 설명: [[원문/01 올인원 가이드]]

## 사례·자료 색인

- [[원문/02 프롬프트와 예시대화]] — 완성 프롬프트, 시스템 구현, 묘사·문체, 예시대화
- [[원문/03 미디어와 AI 이미지 제작]] — 이미지 생성, 태그, 후처리, 미디어 출력
- [[원문/04 키워드북 테스트 프로필 커스텀]] — 키워드북, 테스트, 프로필, 유저 노트, 커스텀 템플릿
- [[원문/05 사용자 팁과 도움받을 곳]] — 사용자 팁, 오류 대응, 커뮤니티 자료

## 검색 요령

1. `Ctrl+Shift+F`로 원하는 기능을 검색합니다. 예: `호감도`, `상태창`, `문체`, `출력`, `이미지`.
2. 검색 결과의 게시글 번호와 제목으로 적합한 사례를 고릅니다.
3. 상세 프롬프트가 필요한 색인 항목은 노트 상단의 Notion 원문 링크에서 해당 접기를 엽니다.
4. 채택한 내용은 새 노트에 복사한 뒤 출처 번호를 남깁니다.
