# Project: DeusXmakina - Lotto Number Generator

## Overview
A simple, modern, and interactive Lotto Number Generator built using vanilla HTML, CSS, and JavaScript. This project follows modern web standards and focuses on providing a clean user experience.

## Detailed Outline
- **Frontend:** HTML5, CSS3 (Baseline features), and Vanilla JavaScript.
- **Features:**
    - Generate random lotto numbers.
    - Responsive design.
- **Structure:**
    - `index.html`: Main entry point.
    - `style.css`: Modern styling.
    - `main.js`: Logic for number generation and UI updates.
    - `GEMINI.md`: AI development guidelines.
