# 파일 명명 규칙

- 상위 폴더는 작업 순서를 나타내는 두 자리 번호를 사용합니다.
- 작품 폴더명은 공개 예정 작품명 또는 고유 작업명을 사용합니다.
- 등장인물 파일은 `캐릭터명.md`로 만듭니다.
- 테스트 ID는 `TEST-0001` 형식으로 증가시킵니다.
- 결정 기록은 `ADR_0001_주제.md` 형식을 사용합니다.
- 프롬프트 버전은 `v0.1.0` 형태의 의미 버전을 사용합니다.
- `최종`, `진짜최종`, `수정본` 같은 이름은 사용하지 않습니다.
