## 기본 작업 흐름

1. 작품 템플릿을 복제합니다.
2. 작품 개요와 핵심 재미를 사용자와 확정합니다.
3. 세계관·등장인물·시스템 SSOT를 작성합니다.
4. 프롬프트 `v0.1.0`을 만들고 테스트 케이스를 작성합니다.
5. 테스트 결과와 변경 기록을 연결하며 버전을 올립니다.
6. 공개 직전에만 공개 패키지를 완성합니다.
